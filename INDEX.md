# Obsidian Engineering Brain

## Purpose
- Use this vault as durable operating memory for software engineering work across multiple projects.
- Keep project-local context close to each project.
- Keep cross-project execution in workstreams.
- Promote reusable knowledge and standards to shared folders.

## Start Here
- Read [[AGENT_PROTOCOL]] before using an AI agent in this vault.
- Drop raw capture into [[00-inbox/README]].
- Create project notes from [[05-templates/template-project]].
- Create cross-project initiatives from [[05-templates/template-workstream]].

## Working Rules
- If context belongs to one project, keep it inside that project folder.
- If work touches multiple projects, give it a home in `02-workstreams`.
- If knowledge is reusable, promote it to `03-shared-knowledge`.
- If a decision applies beyond one project, record it in `04-shared-decisions`.
- Sessions live near the work they describe, not in a global dump folder.
