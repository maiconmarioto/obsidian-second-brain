# Vault Map

## Root folders
- `00-inbox/`: raw capture
- `01-projects/`: project-local execution memory
- `02-workstreams/`: cross-project execution memory
- `03-shared-knowledge/`: reusable engineering knowledge
- `04-shared-decisions/`: standards and cross-project decisions
- `05-templates/`: note templates
- `06-ops/`: vault operations and dashboards

## Design intent
- Local stays local.
- Cross-project gets a dedicated home.
- Shared knowledge and standards are promoted out of execution folders.
