# Inbox

## Purpose
- Capture first.
- Organize later.

## What belongs here
- raw notes
- loose ideas
- copied terminal output
- links to investigate
- context snippets from chats
- follow-up reminders that have not been routed

## What should not stay here
- active project plans
- reusable knowledge
- accepted decisions
- durable execution notes

## Exit criteria
- Route project-specific notes into a project folder.
- Route cross-project notes into a workstream.
- Rewrite reusable knowledge into `03-shared-knowledge`.
- Convert durable choices into a decision note.
